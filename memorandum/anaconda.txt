注意先激活p36环境
pip install numpy -i https://pypi.tuna.tsinghua.edu.cn/simple

采用conda 安装python包时，可以使用以下命令：
conda install -c https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/linux-64 numpy
其中参数-c指定了镜像源的通道
conda install -c https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/conda-forge/linux-64 numpy
